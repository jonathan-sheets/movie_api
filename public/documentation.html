Movie API project

Create a REST API with Node.js

Objective

Build the server-side component of a "movies" web application where users will be able to access information about different movies, directors, and genres. Users will also be able to sign up, update their personal information, and create a list of their favorite movies.

Request Request Method URL Endpoint Response
Get a list of all movies GET /movies JSON
Return data about a single movie by title GET /movies/:Title JSON
Return data about genre of a single movie by title GET /movies/genres/:Title JSON
Return data about a director by name GET /movies/directors/:Name JSON
Allow new users to register POST /users JSON
Allow users to update their user info (username) PUT /users/:Username JSON
Allow users to add a movie to their list of favorites POST /users/:Username/favorites/:movieID Message (text)
Allow users to remove a movie from their list of favorites DELETE /users/:Username/favorites/:movieID Message (text)
Allow existing users to deregister DELETE /users/:Username Message (text)
