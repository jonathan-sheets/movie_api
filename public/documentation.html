Movie API project

Create a REST API with Node.js

Objective

Build the server-side component of a "movies" web application where users will be able to access information about different movies, directors, and genres. Users will also be able to sign up, update their personal information, and create a list of their favorite movies.

Request Description URL Endpoint Request Method URL Query Parameters Request Body Response Format
Get a list of all movies /movies GET /movies None JSON
Return data about a single movie by title /movies/Terminator GET /movies/:title None JSON
Return data about a genre by name /movies/genres GET /movies/genres/:name None JSON
Return data about a director by name /movies/directors/James%20Cameron GET /movies/directors/:name None JSON
Allow new users to register /users POST /users JSON object holding user data:

{
Username: kylereese,
Password: password,
Email: example@abc.com
Birthday: YYYY/MM/DD
}

JSON
