flixNET API project:

Create a REST API with Node.js

Objective:

Build the server-side component of a "movies" web application where users will be able to access information about different movies, directors, and genres. Users will also be able to sign up, update their personal information, and create a list of their favorite movies.

Request Method Request Description URL Endpoint URL Query Parameters Request Body Response Format
GET Get a list of all movies /movies /movies None JSON
GET Return data about a single movie by title
(use %20 for spaces)		 /movies/Terminator /movies/:title None JSON
GET Return data about a genre by name /movies/genres/Action /movies/genres/:name None JSON
GET Return data about a director by name
(use %20 for spaces)		 /movies/directors/James%20Cameron /movies/directors/:name None JSON
POST Allow new users to register /users /users JSON object holding user data:
{
Username: kylereese,
Password: password,
Email: example@abc.com
Birthday: YYYY/MM/DD
}
JSON
POST Allow users to add a movie to their list of favorites /users/kylereese/favorites/ /users/:Username/favorites/:movieID JSON object
{
"FavoriteMovies": ["5f9ae0a93bc461d6cbc26250"]
} 		JSON
PUT Allow users to update their user info (username) /users /users/:Username JSON object holding user data:
{
Username: kylereese,
Password: password,
Email: example@abc.com
Birthday: 01/07/1986
}
JSON
DELETE Allow users to remove a movie from their list of favorites /users/kylereese/favorites/ /users/:Username/favorites/:movieID JSON object
{
"FavoriteMovies": ["5f9ae0a93bc461d6cbc26250"]
} 		JSON
DELETE Allow existing users to deregister /users/kylereese /users/:Username None Message (text)
